About

Limitations

  • Only works in cities within the United States.

  • Only has three time periods (one day, one week, and two weeks) for forecasts.

  • Humidity data is not available for time periods longer than one day.

  • Only temperature, humidity, and wind speed weather data is available in this app.

OpenMeteo API

This project uses the OpenMeteo API for geolocation and tracking. The geolocation API translates search queries into GPS coordinates, and the weather API turns those GPS coordinates into real weather data.

Quickcharts API

This project uses the Quickcharts API to physically represent the data that came from the quickcharts API. It takes a get request with datasets and labels specified in its parameters.
